Event tabs change events:

Go to the scoreboard on the top right of the screen.

By official's names:

  • Red dot: official not logged in
  • Green dot: official logged in

To adjust Start List:

  • Scratch an event, uncheck () the gymnast's name.
  • Rearrange start list: click and drag the icon () beside the gymnast's name.
  • Swap a gymnast's analysis with another gymnast, click the gear icon () next to the name of the gymnast who was assigned the score.
  • Remove a gymnast from an event, click Reset Current gymnast
      • Note: deleted routines cannot be recovered.

To message anyone in the competition, open the chat window () on the bottom right of the screen.

Test Mode:

  • Test mode is on by default. This allows you to test cameras and judges. This must be disabled before the start of the competition.
  • Disabling test mode resets the meet to the start and clears out routines and scores.
